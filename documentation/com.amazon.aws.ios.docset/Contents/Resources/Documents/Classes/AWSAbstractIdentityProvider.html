Overview

An abstract implementation of the AWSCognitoIdentityProvider. Developers should extend this class when they want to implement developer authenticated identities but do not need to support the basic Amazon Cognito authflow in the same application.

getIdentityId

Get/retrieve the identity id for this provider. If an identity id is already set on this provider, no remote call is made and the identity will be returned as a result of the BFTask (the identityId is also available as a property). If no identityId is set on this provider, one will be retrieved from the service.
